# Default Music Collection

This folder contains the default music tracks for each emotion category. These will be used when automatic emotion-based music selection is enabled.

## Music Categories:
- `happy/` - Upbeat, joyful tracks
- `sad/` - Melancholic, emotional tracks
- `angry/` - Intense, powerful tracks
- `neutral/` - Balanced, ambient tracks
- `excited/` - High-energy, dynamic tracks

Add your own music files to these folders to expand the collection. The system will randomly select an appropriate track based on the detected emotion.
